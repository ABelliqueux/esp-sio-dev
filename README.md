## SD CARD
-------
### ESP to SD Card (SPI Mode)
* 21    CS
* 22    MOSI // 10K resistor pullup to 3.3v
* 19    CLK/SCK
* 23    MISO
* GND   Ground
* 3.3   VDD

## PS1
-------
### ESP to Playstation
* 32    1 DATA
* 34    2 CMND
* 35    6 ATT
* 39    7 CLK
* 33    8 ACK
